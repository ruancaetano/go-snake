# Go Snake
Snake game built with Go and [Termloop](https://github.com/JoelOtter/termloop).

<img src="./docs/demo.gif" width=400>

## How to play?

- Install: `go get -u github.com/ruancaetano/go-snake`
- Play it: `$GOPATH/bin/go-snake`
